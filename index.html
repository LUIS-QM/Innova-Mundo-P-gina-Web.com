Páginas web

Una página web es un documento electrónico que es capaz de contener: texto, imágenes, sonido, vídeo, enlaces, programas… Cada una de las URL que encontremos en un sitio web, son páginas de internet.

Sitios Web

Un sitio web o cibersitio es aquél en el que encontramos una colección de páginas web relacionadas y comunes a un dominio de internet. Cada vez que utilicemos una World Wide Web o ‘www’ específica.

Seo de Sitios Web

Es el posicionamiento en buscadores u optimización de motores de búsqueda es el proceso de mejorar la visibilidad de un sitio web en los resultados orgánicos de los diferentes buscadores. También es frecuente nombrarlo por su título inglés, SEO (Search Engine Optimization).
